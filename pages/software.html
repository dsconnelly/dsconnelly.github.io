I maintain a few small Python packages.

  • mubofo implements boosted forest regression models that natively support multioutput problems. At present, it's reasonably performant, but a long-term goal of mine is to speed it up by rewriting the backend entirely in C.
  • func2cli allows automatic generation of command line interfaces of Python scripts, provided the functions they call have appropriately formatted docstrings.

In addition, I'm the creator of Escucho, a simple web application that lets you practice transcribing Spanish-language audio. More features and more languages to come!
